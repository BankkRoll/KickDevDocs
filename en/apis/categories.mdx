# Categories

Categories APIs provide information about the content categories used on Kick, including games and special categories like "Just Chatting".

<Card title="Get Categories" icon="list" method="GET" path="/categories" api="GET /categories">
  Retrieve a list of all available content categories.
</Card>

<Card title="Get Category" icon="folder" method="GET" path="/categories/{id}" api="GET /categories/{id}">
  Retrieve information about a specific category by ID.
</Card>

## Get Categories

Retrieve a list of all content categories available on the Kick platform.

<Note>
  No authentication is required for this public endpoint.
</Note>

### Query Parameters

<ParamField query="limit" type="integer" default="25" initialValue="5">
  Number of categories to return (max 100)
</ParamField>

<ParamField query="cursor" type="string">
  Pagination cursor for retrieving the next set of results
</ParamField>

<ParamField query="search" type="string">
  Filter categories by name
</ParamField>

### Response

<ResponseField name="categories" type="array">
  <Expandable title="Array of category objects">
    <ResponseField name="id" type="integer">
      Unique identifier for the category
    </ResponseField>
    <ResponseField name="name" type="string">
      Name of the category
    </ResponseField>
    <ResponseField name="slug" type="string">
      URL-friendly identifier for the category
    </ResponseField>
    <ResponseField name="icon" type="string">
      URL to the category icon
    </ResponseField>
    <ResponseField name="banner" type="string">
      URL to the category banner image
    </ResponseField>
    <ResponseField name="viewers" type="integer">
      Current number of viewers watching streams in this category
    </ResponseField>
    <ResponseField name="tags" type="array">
      <Expandable title="Array of tag objects">
        <ResponseField name="id" type="integer">
          Tag identifier
        </ResponseField>
        <ResponseField name="name" type="string">
          Tag name
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="pagination" type="object">
  <Expandable title="Pagination object">
    <ResponseField name="cursor" type="string">
      Cursor for the next page of results
    </ResponseField>
    <ResponseField name="has_more" type="boolean">
      Whether there are more results available
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash
curl --request GET \
  --url "https://api.kick.com/public/v1/categories?limit=5"
```

```javascript
const getCategories = async () => {
  const response = await fetch('https://api.kick.com/public/v1/categories?limit=5', {
    method: 'GET'
  });
  
  const data = await response.json();
  console.log(data);
  return data;
};

getCategories();
```

```python
import requests

params = {
    'limit': 5
}

response = requests.get('https://api.kick.com/public/v1/categories', params=params)
result = response.json()
print(result)
```
</RequestExample>

## Get Category

Retrieve detailed information about a specific category by its ID.

<Note>
  No authentication is required for this public endpoint.
</Note>

### Path Parameters

<ParamField path="id" type="integer" required>
  The ID of the category to retrieve
</ParamField>

### Response

<ResponseField name="category" type="object">
  <Expandable title="Category object">
    <ResponseField name="id" type="integer">
      Unique identifier for the category
    </ResponseField>
    <ResponseField name="name" type="string">
      Name of the category
    </ResponseField>
    <ResponseField name="slug" type="string">
      URL-friendly identifier for the category
    </ResponseField>
    <ResponseField name="icon" type="string">
      URL to the category icon
    </ResponseField>
    <ResponseField name="banner" type="string">
      URL to the category banner image
    </ResponseField>
    <ResponseField name="description" type="string">
      Detailed description of the category
    </ResponseField>
    <ResponseField name="viewers" type="integer">
      Current number of viewers watching streams in this category
    </ResponseField>
    <ResponseField name="tags" type="array">
      <Expandable title="Array of tag objects">
        <ResponseField name="id" type="integer">
          Tag identifier
        </ResponseField>
        <ResponseField name="name" type="string">
          Tag name
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="top_channels" type="array">
      <Expandable title="Array of top channel objects in this category">
        <ResponseField name="id" type="integer">
          Channel ID
        </ResponseField>
        <ResponseField name="name" type="string">
          Channel name
        </ResponseField>
        <ResponseField name="slug" type="string">
          Channel slug
        </ResponseField>
        <ResponseField name="profile_image" type="string">
          URL to channel profile image
        </ResponseField>
        <ResponseField name="viewers" type="integer">
          Current viewers on the channel
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash
curl --request GET \
  --url "https://api.kick.com/public/v1/categories/1"
```

```javascript
const getCategory = async (categoryId) => {
  const response = await fetch(`https://api.kick.com/public/v1/categories/${categoryId}`, {
    method: 'GET'
  });
  
  const data = await response.json();
  console.log(data);
  return data;
};

getCategory(1);
```

```python
import requests

category_id = 1
response = requests.get(f'https://api.kick.com/public/v1/categories/{category_id}')
result = response.json()
print(result)
```
</RequestExample>
